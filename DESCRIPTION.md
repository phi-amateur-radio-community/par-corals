# Par Corals

## About

A Minecraft mod focused on corals and the ocean.

This mod expands Minecraft’s relatively limited coral content by adding new coral-themed features, including:

- Items
- Creatures
- Dimensions
- Devices and utilities
- More ocean-related gameplay content

The goal of this project is to make Minecraft’s ocean ecosystem richer, more vibrant, and more worth exploring.

Downloads, discussions, suggestions, and contributions are always welcome.

This project is part of [PhiARC](https://www.phiarc.org/).
